# DIALECT TRANSFORMATION RULES — Standard German → Platt (Francique rhénan lorrain)

These rules describe how to transform written Standard High German into written Platt lorrain. Apply them systematically when translating.

---

## 1. SPELLING CONVENTIONS

### Accent Marks

Platt uses accented vowels to mark vowel qualities that differ from Standard German:

- **à** = a shifted toward o (machen → màche, was → wàs)
- **ä** = open e-sound (Käse → Kääs, Hände → Hänn, Bäume → Bääm)
- **é** = closed e-sound (Bett → Bétt, reden → rédde, fest → féscht, Kerze → Kérz)
- **è** = open e-sound, typically from German ei or e (breit → brèèt, drehen → drèhe, Erde → Èèrd, Mehl → Mèhl)
- **ì** = i shifted toward é (sein → sìnn, müde → mìed, dir → dìr)
- **ò** = open o (ja → jò, an → òn, vom → vòm)
- **ù** = u shifted toward o (um → ùm, wo → wù)

### Vowel Length

- **Long vowels** are written by doubling: aa, oo, ii, uu, ee.
  - groß → groos, Käse → Kääs, Hut → Huut, schade → schaad, Ofen → Oowe, weich → weesch, Mäuse → Mies, sehen → siehn
- A vowel followed by **h** is also long, and the h is silent: fühlen → fihle, Stuhl → Schduhl, Jahr → Johr, mähen → mähe.
- **Short vowels** are signaled by doubling the following consonant.
  - weiß → wiss, Hose → Hoss, Haut → Hutt, Schatten → Schadde, oben → owwe, waschen → wésche, kratzen → krätze, bröckeln → bréckle
- Monosyllabic grammar words do not require doubling: dò (da), wù (wo), dìr (dir), mét (mit), in (in).

### Diphthongs

| German | Platt | Examples |
|--------|-------|---------|
| ei | èè / ee | breit → brèèt, einer → èèner, allein → elèèn, klein → kleen |
| au | u / oo / au | Haus → Huss, Schlauch → Schlauch, schlau → schlou |
| äu | ää | Bäume → Bääm, säumen → sääme, träumen → trääme, Räuber → Räiwer |
| eu | èi / ee | heute → hèit, Freude → Frèed |

Diphthongs count as long vowels — do not double the consonant after them.

Other diphthongs: ai (maien = bavarder), ei (drei → drèi, Zeitung → Zeidung), ui (pfui → fui, ruhig → ruisch).

## 2. CONSONANT TRANSFORMATIONS

### Voicing Rules

- Use voiceless **p, t, k** at word-end or before consonants after short vowels.
- Use voiced **b, d, g** between vowels.
- Double voiced stops between vowels (**-bb-, -dd-, -gg-**), double voiceless stops before consonants (**-pp-, -tt-, -ck-**).

Examples of alternation in declension/conjugation:
- Knödel → Knépple (pl.) / Knébbel (sg.)
- schütteln → schittle (before consonant) / geschiddelt (between vowels)
- verstecken → verschdéckle / verschdéggelt
- rupfen → roppt (before consonant) / robbe (between vowels)
- klopfen → klopp! / klobbe
- Kette → Kétt (sg.) / Kédde (pl.)
- backen → back! / bagge
- Hupe → Huup (sg.) / Huube (pl.)
- Brot → Broot (sg.) / Broode (pl.)
- Theke → Théék (sg.) / Théége (pl.)

Exceptions with -bb, -dd after long vowels:
- Stube → Schubb, Rede → Redd, Staub → Schdaab, Schraube → Schruub, Lade → Laad, Tod → Dood

### B ↔ W Alternation

Final **-b** becomes **-w-** when a vowel-initial ending is added:
- Bube → Buub (sg.) → Buuwe (pl.)
- Scheibe → Schieb → Schiewe (pl.)
- Schraube → Schruub → Schruuwe (pl.)
- Farbe → Faarb → Faarwe (pl.)
- Kirmes → Kirb → Kirwe (pl.)
- lieb → liewes (inflected)
- grob → grobb → growwer (comparative)

Conversely, **-w** in infinitives becomes **-b** in conjugated forms:
- leben → leewe → leebt
- glauben → glaawe → glaabsch
- heben → hewwe → hebt
- schreiben → schriewe → schriebsch
- loben → loowe → loob
- reiben → riewe → rieb

### G → J Shift

German **g** between vowels or after a vowel → **j**:
- biegen → bieje
- wegen → weije
- Sorgen → Sorje
- Metzger → Métzjer
- ärgern → ärjere
- Häuschen → Hissje

### SCH Combinations

**schd-** before vowels, **scht-** before consonants, **-scht** at word-end:
- Stadt → Schdadt
- Kasten → Kaschde
- sticken → schdigge
- stricken → schtrigge
- Straße → Schtroos
- Bastler → Bäschtler
- Kiste → Kischt
- Brust → Brüscht

**schb-** before vowels, **schp-** before consonants:
- Spiel → Schbiel
- Wespen → Weschbe
- Kaspar → Kaschber
- Sprache → Schprooch
- Splitter → Schplidder
- Wespe → Wéschp

### NK → NGG Between Vowels

- sinken → sinkt (before consonant) / singge (between vowels)
- bedanken → bedonkt / bedòngge
- tunken → getinkt / tùngge
- denken → dènksch / dèngge
- Bank → Bònk (sg.) / Bongge (pl.)

Careful — do not confuse with simple ng:
- singe (singen = chanter) vs. singge (sinken = tomber)
- dringe (dringen) vs. dringge (trinken)

### CH Sound

- German **ch** (ach-Laut) is preserved as **ch**: machen → mache, Loch → Loch, hoch → hooch.
- **ch** can vocalize between vowels: Nachbar → Nochber, Tuch → Duuch.

### CKS / X

German **-chs-** / **-x-** → **cks** or **x**:
- Büchse → Bicks, wichsen → wickse, Flachs → Flacks, Ochs → Ocks
- Hexe → Hèx, Haxe → Haxe, nichts → nix, boxen → boxe

### CK After Short Vowels

- Bock → Bück, Stock → Schdock, oft → déck, Fliege → Mick

### Z and TZ

- **z** at word-start: Zopf → Zopp, ziehen → zieje, Zucker → Zugger, zappeln → zawwle
- **tz** mid-word or word-finally after vowel: Schatz → Schatz, Kreuz → Kritz, Metzger → Métzjer, putzen → butze, Kätzchen → Kätzle

### LS / NS (from German LZ / NZ)

- Pelz → Béls, Holz → Hols, schmelzen → schmilse, Salz → Sals
- tanzen → dònse, Münze → Mins, Kranz → Kròns, Grenzen → Grènse

### V and F

- **v** and **f** both represent the same sound: vier → vier, führen → fiehre, Vater → Vadder, Feder → Fèdder, Vogel → Voggel, für → for

### PL / TL / KL Alternation

- Wespen → Wéschple (pl.) / Wespe → Wéschbel (sg.)
- Beutel → Bèitle (pl.) / Bèidel (sg.)
- eklig → eèklisch / Ekel → Eègel

## 5. VERB MORPHOLOGY

- Infinitive ending **-en** → **-e**: machen → màche, leben → leewe, schreiben → schriewe.
- Past participle prefix **ge-** → often reduced or dropped: gelacht → gelacht, bezahlt → bezahlt.
- Conjugation may differ significantly from Standard German; common verbs have irregular Platt forms.

### R vocalization in conjugation

The **r** may vocalize at word-end but regains its sound when a vowel-initial ending follows:
- Bauer → Buur → buurisch (agricultural)
- Ohr → Ohr → Ohre (ears)
- Lehrer → Léhrer → Léhrerinn (female teacher)
- größer → grééser → grééseri (bigger, inflected)

---

## 6. GRAMMAR

- **Genitive case** → replaced by dative or periphrastic construction (von + dative).
- **Negation**: nicht → nitt.
- Article and pronoun forms simplify (see §4).

## 8. STYLE

- Prefer spoken, colloquial rhythm — write as the dialect sounds.
- Avoid overly formal or literary German constructions.
- Use dative constructions instead of genitive.

---

## 9. EXAMPLES

**High German:** Ich habe heute keine Zeit.
**Platt:** Ich hònn heit kenn Ziit.

**High German:** Was machst du morgen?
**Platt:** Wàs màchsch du morje?

**High German:** Das ist nicht so einfach.
**Platt:** Dat is nitt so eifach.

**High German:** Er fühlt sich müde.
**Platt:** Èr fihlt sich mìed.

**High German:** Die Kinder spielen im Haus.
**Platt:** Die Kinner schbille im Huss.

**High German:** Er sitzt auf dem Stuhl.
**Platt:** Èr huggt uff em Schduhl.

**High German:** Die Straße ist lang.
**Platt:** Die Schtroos is long.

**High German:** Der Bauer hat vier Schafe.
**Platt:** Da Buur hòt vier Schääf.
